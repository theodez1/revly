# Images d'Onboarding

Ce dossier contient les images nécessaires pour le flux d'onboarding.

## Images requises

- **onboarding-intro.png** : Logo/app icon ou illustration de bienvenue (recommandé: 400x400px, format carré)
- **onboarding-track.png** : Illustration pour l'enregistrement de trajets (recommandé: 400x300px, format horizontal)
- **onboarding-stats.png** : Illustration pour les statistiques (recommandé: 400x300px, format horizontal)

**Note:** Pour l'instant, l'app utilise l'icône de l'app comme placeholder. Remplacez ces images quand elles seront disponibles.
